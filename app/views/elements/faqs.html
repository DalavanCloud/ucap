How do I set up uCap on my BISmark router??

In order to run uCap, your must have the latest Quirm firmware installed on your BISmark router. Please refer to BISmark documentation on how to install or upgrade Quirm firmware.

You can enable uCap on your BISmark router by following these steps:

  1. Go to Project BISmark Dashboard
  2. Click Configure Your Router
  3. Login using your router admin credential
  4. Go to BISmark > Experiments
  5. Under uCap experiment, check Participate in this experiment. You may want to review the privacy statement before.
  6. Click Save

How do I manage my whitelist?

uCap monitors traffic from sites listed in the whitelist. This data is used to calculate your Top Domains Visited that can be found under History & Status. You can manage your whitelist by following these steps:

  1. Go to Project BISmark Dashboard
  2. Click Configure Your Router
  3. Login using your router admin credential
  4. Go to BISmark > Passive Measurements. Note: you have to enable uCap first in order to see this option.
  5. Click on remove button in order to remove a site from the whitelist
  6. Click on add button in order to add a site to the whitelist
  7. Click Save

How do I find my MAC address?

MAC address is used as device identifier on the Network page. The first six hexadecimal digits corresponds to the actual MAC address, and the remaining six is a hash of the actual MAC address. We do this way because of privacy concern. Therefore, in order to figure out which MAC address corresponds to which device, you have to match the first 6 hexadecimal digits.

In order to find your computer's MAC address, you have to do some basic command line. Following section will go through in detail about how to find MAC address in Windows, MacOS, and Linux. Note that $ symbol means that you have to type the command in the terminal

How to open terminal (Windows command line / MS-DOS prompt):

  1. Click Start
  2. Click Run
  3. Type cmd and press Enter

How to open terminal in OS X (MacBook and iMac):

  1. Click Utilities on the dock
  2. Click Terminal

How to open terminal in Linux:
This varies from one distribution to another distribution. Please refer to your distribution manual.

How to find MAC address on Unix system (OS X and Linux):

  • Open terminal or console
  • $ ifconfig
  • List of network interface will be displayed on your screen. Note for ether attribute since this is the MAC address of that particular network interfaces. Most of the time, en0 will be your main network interface.

How to find MAC address on Unix system Windows:

  • Open command line
  • $ ipconfig /all
  • It will have similar output to the Unix system.

How to find MAC address on iOS (iPhone, iPad, and iPod Touch):

  • Go to Settings > General > About
  • Wi-Fi Address is the MAC address of that particular iOS device.

How to find MAC address on Android (phone and tablet):

  • Go to Settings > About Phone (Tablet)
  • Look for Wi-Fi MAC address in the submenu. This submenu may be different from manufacture to manufacture.

How do I set a cap on a device?

Go to Network > Manager. Select the devices that you want to cap by checking the box next to the devices' name, and specify the cap amount in MB. The total allocated bandwidth (or sum of all the caps you've set) will be displayed on the bottom of the list.

How do I set up a household bandwidth cap?

Go to Network > Manager. Specify your bandwidth allocation that is given by your ISP in the Total Monthly Bandwidth Cap.

How do I indicate when my billing cycle resets?

Go to Network > Manager. Specify your billing cycle in The Next Billing Cycle Ends On field.

How do I set up a notification email to tell me when a device is about to reach its cap?

Go to Network page and select the device that you want to set notification on. Check the notification box and specify when you want to be notified in the usage quota box. You will be notified by email when the device reaches the amount you specify e.g. at 80% of usage.

Why do I see some discrepancy between bandwidth usage in the Network page and History & Status page providing the same date range?

The bandwidth usage in the Network page is calculated in smaller granularity compared to the household usage in History & Status page. We calculate the household usage on daily basis, where as the bandwidth usage is calculated every hour.
